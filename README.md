# vibe-coding-mission
3개월 내 AI 기반 수익화 모델 구축 미션 (Goal: 월 매출 100만 원). 바이브 코딩(Vibe Coding) 철학 아래 Jules, Cursor, Claude, Antigravity IDE 등 최신 AI 에이전트와 IDE 생태계를 총동원한 비즈니스 자동화 서비스 개발 프로젝트.

① Tech Stack (기술 스택)
AI가 프로젝트에 적합한 라이브러리를 제안할 수 있도록 명시합니다.

Frontend: Next.js (App Router), Tailwind CSS

Backend/Database: Supabase (Auth, DB, Storage)

Deployment: Vercel

AI Integration: Gemini API, OpenAI API

② Agent Collaboration Guide (에이전트 협업 안내)
에이전트들이 자신의 역할을 잊지 않도록 경로를 지정합니다.

"상세한 에이전트 협업 규칙 및 역할 분담은 AGENTS.md 파일을 참조할 것."

③ Project Roadmap (90일 수익화 로드맵)
3개월 미션의 단계를 명시하여 현재 작업의 우선순위를 알립니다.

Phase 1 (Day 1-30): MVP(최소 기능 제품) 구축 및 초기 배포

Phase 2 (Day 31-60): 사용자 피드백 수집 및 유료 모델(결제) 적용

Phase 3 (Day 61-90): 마케팅 자동화 및 월 매출 100만 원 달성
